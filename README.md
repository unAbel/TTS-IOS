# English Word Learner - Text-to-Speech App

English Word Learner es una aplicación de iOS que te ayuda a aprender nuevas palabras en inglés utilizando la tecnología Text-to-Speech (TTS). Con esta aplicación, puedes escuchar la pronunciación correcta de las palabras en inglés y mejorar tu vocabulario de una manera interactiva y divertida.

## Características

- **Text-to-Speech (TTS)**: La aplicación utiliza la función de Text-to-Speech de iOS para convertir palabras en texto a voz, permitiéndote escuchar la pronunciación de las palabras en inglés.
- **Banco de palabras**: La aplicación incluye un banco de palabras con una amplia variedad de vocabulario en inglés, organizado por categorías como alimentos, animales, colores, etc.
- **Función de búsqueda**: Puedes buscar palabras específicas dentro del banco de palabras para encontrar pronunciaciones y significados.


## Requisitos del Sistema

- iOS 16.2 o superior.
- Dispositivo iPhone o iPad.

## Instalación

1. Clona este repositorio en tu máquina local.
2. Abre el archivo `.xcodeproj` en Xcode.
3. Compila y ejecuta la aplicación en tu dispositivo iOS o en el simulador.
